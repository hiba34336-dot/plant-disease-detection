# Plant Disease Detection App

A Machine Learning-powered web application that detects plant diseases from leaf images using Deep Learning and Computer Vision techniques. The app helps farmers and users identify plant diseases quickly and accurately through an easy-to-use interface.

---

## Features

- Upload plant leaf images
- AI-powered disease prediction
- Fast and accurate results
- User-friendly web interface
- Deep Learning image classification model
- Supports plant disease identification in real time

---

## Technologies Used

- Python
- TensorFlow / Keras
- CNN (Convolutional Neural Network)
- OpenCV
- NumPy
- Streamlit / Flask
- Machine Learning
- Deep Learning
